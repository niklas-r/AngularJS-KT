What is MVC?

You've probably heard of MVC before. It's a way to structure your application.

Although it's wide spread in other programming languages, it's still quite new in JavaScript.

Model

The data in raw form, usually JSON och XML in JavaScript scenarios. It could look something like this:


{
  "users" : [
    {
      "firstName": "John",
      "lastName": "Doe",
      "id": "0"
    },
    {
      "firstName": "Jane",
      "lastName": "Doe",
      "id": "1"
    }
  ]
}

And so on...

What is AngularJS?

JS MV* Framework used to (mainly) creating Single Page Applications.

Also great for creating encapsulated "components".

Angular gives you structure

The Controller

No code up until now.

The controller is a constructor function used to setup initial data and add behaviour to the view.

What are scopes?

The scope is the "glue" between your controller and your view.


function GreetCtrl($scope) {
  $scope.name = 'World';
}

function ListCtrl($scope) {
  $scope.names = ['Igor', 'Misko', 'Vojta'];
}

Let's get a bit more scientific.

It's an execution context for expressions.

On instantiation a single scope is created, $rootScope.

All scopes inherit prototypically.

New scopes are created each time you use one of these:

  • ng-include
  • ng-switch
  • ng-repeat
  • ng-view
  • ng-controller
  • directives
